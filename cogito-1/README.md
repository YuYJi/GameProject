# Cogito-1: 仿人类大脑 AI 系统

Cogito-1 是一个模仿人类大脑认知过程的人工智能系统，采用模块化分层设计，旨在实现更接近人类思维模式的智能行为。

## 设计理念

- **模块化分层设计**：将认知过程分解为感知、工作记忆、长时记忆和元认知等独立模块
- **有限容量工作记忆**：模拟人类 4±1 组块的工作记忆容量
- **多模态感知**：支持处理多种输入模态，如视觉、听觉、语言等
- **动态记忆系统**：包含语义记忆（知识图谱）、情景记忆（事件序列）和内隐记忆（技能库）
- **情感与元认知**：模拟人类的情绪反应和自我认知能力
- **记忆巩固与遗忘**：实现基于艾宾浩斯遗忘曲线的记忆管理

## 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动系统

```bash
python scripts/run_cogito.py
```

### 运行测试

```bash
python -m pytest tests/
```

## 配置说明

系统配置文件位于 `config/` 目录，包含以下主要配置：

- `default.yaml`：默认系统参数
- `logging.yaml`：日志配置
- `model_params/`：各子系统的超参数
